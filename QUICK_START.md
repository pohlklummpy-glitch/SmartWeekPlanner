## 🏗️ Option 2: Development Build (Alle Features)

### Voraussetzungen:
- ✅ Expo Account (kostenlos)
- ✅ Apple Developer Account (99€/Jahr) **ODER** AltStore (kostenlos)

### Schritt 1: EAS CLI installieren
```bash
npm install -g eas-cli
```

### Schritt 2: Anmelden
```bash
eas login
```

### Schritt 3: Build starten
```bash
eas build --profile development --platform ios
```

### Schritt 4: Warten (15-30 Min)
```bash
# Status checken:
eas build:list
```

### Schritt 5: Installieren

#### Mit Apple Developer Account:
1. Download `.ipa` Datei
2. Xcode → Devices → Ziehe `.ipa` auf iPhone

#### Mit AltStore (KOSTENLOS):
1. Download `.ipa` Datei
2. AltStore installieren: https://altstore.io
3. AltStore → "+" → Wähle `.ipa`
4. ⚠️ Alle 7 Tage neu signieren
